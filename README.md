**GoDraw** is a free, open-source, and feature-rich digital whiteboard built with love in India. It serves as a powerful alternative to popular tools like **Excalidraw**, **tldraw**, and **Miro**, with the mission to provide a fast, reliable, and accessible tool for students, teachers, designers, and teams everywhere.

Our goal is to create a world-class collaborative tool that is both powerful for professionals and simple for everyday use.

## üöÄ Getting Started (For Developers)

Want to run GoDraw on your local machine? Follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rishabh21g/godraw.git
    cd godraw
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

    The application will be available at `http://localhost:5173`.
